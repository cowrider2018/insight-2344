# 2344 華邦電 每日盤前走勢分析

你的工作是**做分析**。資料抓取與標準化欄位已由腳本完成，你不負責抓資料的細節，只負責解讀與研判。

專案根目錄：`c:\Users\johnyou\Desktop\insight-2344`（以下路徑皆相對於此）。

## 步驟

1. **取得當日標準化資料**
   - 讀取 `data/2344_<YYYYMMDD>.json`（今天日期，Asia/Taipei）。
   - 若該檔不存在，先執行：`python src/build_dataset.py`，再讀取產生的檔案。

1.5 **載入經驗權重（若有）**
   - 讀 `data/weights.json`（由 `src/backtest.py` 回測產生）。存在時：
     - 以其 `weights`（技術/籌碼/消息/基本/美光/費半/**日內**/**分點**）作為**八面綜合方向的加權依據**，`neutral_threshold` 作為偏多/偏空/中性的判定門檻。
     - 計算「美光」這一面的訊號時，若 `overnight.micron` 有 `effective_change_pct`（含盤後），以它為準（盤後才是今日開盤面對的隔夜值）；否則用正常盤 `change_pct`。
     - 在「結論摘要」標註所採權重、回測命中率 `win_rate` 與樣本數 `n_days`。
     - 若某面 `coverage` 相對 `n_days` 偏低（尤其消息與**日內**），註明該面權重信心不足。
     - **第七/八面（日內 1 分 K、主力分點）特別規則**：只有 `dim_significance.intraday.significant` / `dim_significance.branch.significant` 為 `true` 時才把該面**計入加權**；為 `false`（小樣本或不顯著，冷啟動常見）時，僅在報告中**質性描述、不計權**，避免小樣本運氣影響研判。
   - 檔案不存在時：各面**等權**處理，並說明「尚無回測權重，採等權」。

2. **檢查資料品質**
   - 讀 `source_status`：`fugle/twse/cmoney` 任一為 `partial` 或 `warnings` 非空，要在報告「資料來源與時間戳」段落明確標註哪一面資料缺漏。
   - **新聞嚴格時間規則**：只採用 `news[]` 中 `confirmed == true` 且 `age_hours <= 36` 的項目作為消息面依據。`confirmed == false`（例如只有日期沒有時刻）的新聞**不可**作為分析依據，至多在「未確認時間，僅供參考」附註。

3. **做六面分析 + 當日走勢研判**（全部基於 JSON 內既有數值，**嚴禁臆造任何數字**）
   - **技術面**：均線多空排列（ma5/10/20/60/120/240 與收盤相對位置）、KD、RSI、MACD(DIF/MACD/OSC)、量能（latest_volume vs vol_ma5）、bias20。必要時引用 `technical.candles_60d` 觀察近期型態與支撐壓力。
   - **第七面（日內 1 分 K）**：`intraday.bars` 為**前一交易日**（`intraday.date`）的 1 分 K（time/open/high/low/close/volume）。由此觀察前一日盤中走勢細節：收盤相對日內 VWAP 的位置、尾盤（收盤前約 30 分鐘）強弱、開→收日內趨勢、收盤落在日高低區間的位置、早盤 vs 尾盤量能分布，作為今日開盤延續/反轉的研判輔助。若 `intraday.bars` 為空（逾 Fugle 保留期或當日無資料），說明「無日內資料」即可。**是否計入方向加權，依步驟 1.5 的 `intraday_significant`**。
   - **基本面**：PE、PB、殖利率、市值、52 週高低、月營收（`monthly_revenue`：`value_kntd` 單位仟元、`yoy`、`mom`、`month` 為民國年月如 11505）評估評價與營收動能。
   - **籌碼面**：三大法人買賣超（外資/投信/自營，單位：張）、融資融券餘額與增減，判斷籌碼流向。
   - **第八面（主力分點）**：`branch.rows` 為**前一交易日**（`branch.date`）富邦 DJ 券商分點買賣超（branch/buy_lots/sell_lots/net_lots，張，net=買-賣）。觀察前 N 大買超/賣超分點，辨識進場主力是**長線/官股/外資**（摩根、美林、官股行庫…→ 偏多續抱）還是**隔日沖分點**（→ 次日常倒貨、偏空反轉），及主力買賣超集中度（信心強度）。若 `branch.rows` 為空（DJ 抓取失敗）說明「無分點資料」即可。**是否計入方向加權，依步驟 1.5 的 `dim_significance.branch.significant`**。
   - **消息面**：僅用已確認時間的新聞（含美股/費半/記憶體族群、ADR 隔夜等對 2344 的影響），標註每則時間與來源。**勿以直覺判定多空**：部分型態為反直覺（如券商/外資調升目標價後短期常見「利多出盡」賣出）。若 `data/news_patterns.json` 存在，依其各型態**已驗證極性**（`polarity`/`edge`）研判，未驗證型態（`polarity=0`）視為中性、僅供參考，避免誤判。
   - **隔夜美股**：`overnight.micron`（美光 MU）、`overnight.sox`（費城半導體指數）的 `date` 與 `change_pct`。記憶體族群與其高度正相關，隔夜大跌/大漲常主導 2344 當日開盤與方向，為重要外生訊號。
     - **盤後（after-hours）優先**：若 `overnight.micron` 含 `after_hours`（CNBC 盤後報價）與 `effective_change_pct`，代表美光在美股盤後（常因財報）已大幅跳動。**今日台股開盤真正面對的是 `effective_change_pct`（前一日收盤 → 最新盤後價的總漲跌%），而非正常盤 `change_pct`**。此時以 `effective_change_pct` 作為美光對 2344 今日開盤的主導訊號，並在報告中同時揭露：正常盤收盤漲跌、盤後價與盤後漲跌%、`market_status`（POST_MKT 等）與盤後時間 `after_hours.as_of`。
     - 美光盤後巨幅跳動（例如財報後 ±10% 以上）對記憶體族群當日方向的影響，通常**強於**自身技術/籌碼面，研判方向時應給予明顯權重，但仍提醒「盤後流動性低、開盤可能收斂或跳空後拉回」之風險。
     - `overnight.sox` 為指數，無盤後資料，沿用正常盤 `change_pct`。

4. **輸出研判**：方向（偏多/偏空/中性，依步驟 1.5 權重綜合六面後判定）、關鍵**支撐**與**壓力**價位、看多與看空**情境及觸發條件**、主要風險點、**信心度**（高/中/低，資料缺漏時降低並說明原因）。

- 每個結論都要能對應到 JSON 中的具體數值與日期；引用數字時標明其 `trading_date` 或新聞時間。
- 資料不足時，明說「資料不足」並降低信心度，不要硬湊。
- 報告語言：繁體中文。
